# Don't-Stop-Me-Now!
Don't Stop Me Now! is My Very First Game made in pure code.  
This game has been made in Python with the Pygame Library.  
You can download this game here or from my itch.io page [osaidii.itch.io](https://osaidii.itch.io)  
Don't Stop Me Now is a racing game where you race against a Computer Car and try to win the races.    

Controls:  
WASD To Move

  
The source code for the gmae is also present in this same repository! 
Please use a venv with the libraries mentioned in requirements.txt
